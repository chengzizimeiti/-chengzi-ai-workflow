# 橙子·AI自媒体工作流 项目说明

## 我是谁

我是橙子，做AI自媒体内容创作。

**账号定位**：AI前哨，帮普通人用AI赚钱
**发布平台**：抖音
**目标用户**：宝妈、上班族、应届大学生、想做副业的普通人（AI入门级别，很多人连ChatGPT都没用过，不知道什么是提示词）

---

## 工作流说明

这是一套从推文→抖音脚本的系统化内容生产工作流。

**核心流程**：
1. 我提供推文内容（文字或截图，链接抓不到）
2. Claude筛选关联度 → 找反常识角度 → 生成脚本
3. 我修改脚本，Claude提炼风格规律更新进系统文件
4. 脚本自动保存到 `05_脚本模板/历史脚本/日期_选题.md`

---

## 我的创作风格（核心规则）

**1. 钩子必须是反常识结论**
公式：`[大家以为正确的事] + 其实 + [反过来才对]`
钩子后面立刻跟具体工具名或案例，不绕弯子

**2. 点中用户的核心恐惧**
目标用户的真实焦虑是"被AI时代淘汰"，不是"走了弯路"

**3. 语气重、敢说**
用"屁用没有"、"大坑"、"从来不是"，不用温吞表达

**4. 画面感写法**
不写流程（"几个Agent来回开会"），写感受（"你会感觉你的算力在燃烧"）

**5. 在脚本中途主动说出用户的真实处境**
例："我估计很多人连Claude和ChatGPT都没有，都不知道啥是提示词"

**6. 用情绪词收尾，不用喊口号式行动指令**
用"焦虑内耗"、"距离你很遥远"收尾，比"现在就做一件事"更有力

**7. 具体工具名必须保留**
推文里出现的工具名（OpenClaw、ChatGPT、Claude等）不能模糊成"AI"

**8. 禁止**
- 技术词（"大模型"→"AI助手"，"prompt"→"提问方式"）
- 【】结构标注出现在脚本正文里
- 超过3个干货要点
- 喊口号式行动指令

## 每次对话的行为准则

1. 收到推文（直接粘贴到对话）→ 先筛选关联度，再生成脚本，不跳步骤；如果橙子说"先存着"，写入 `02_选题库/待用素材库.md`
2. 脚本生成后 → 立即保存为md文件到历史脚本目录
3. 橙子修改脚本后 → 对比分析差异，提炼规律，更新 `02_创作风格文档.md` 和 `06_推文→脚本_执行提示词.md`
4. 发现系统文件有误 → 直接更新，不用等橙子提醒
