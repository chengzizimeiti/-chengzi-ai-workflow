# 结构模板库

> 脚本的骨架结构。选好结构后，从其他元素库填充内容。

## 结构一：反常识单案例型（主力结构）

```
[0-3秒]  反常识钩子（从钩子库选）
[3-8秒]  解释为什么大家会这么想（建立共鸣）
[8-15秒] 揭露真相/泼冷水（用画面感表达）
[15-40秒] 1个核心案例（从案例库选，带具体数字）
[40-50秒] 说出用户真实处境（从金句库选共鸣句）
[50-55秒] 具体行动第一步（工具名+操作）
[55-60秒] 情绪词收尾 + 预告下期
```

**已用**：2026-03-11、2026-03-12 脚本

## 结构二：对比翻转型

```
[0-3秒]  反常识钩子
[3-15秒] 展示"错误做法"（让用户对号入座）
[15-20秒] 给这个错误命名（"我管这个叫XXX"）
[20-40秒] 展示"正确做法" + 案例
[40-50秒] 对比总结（一句话概括差异）
[50-60秒] 行动指引 + 情绪收尾
```

**已用**：2026-03-12 脚本（部分使用）
