# 路径二结构模板（AI小知识系列）

> 每天讲清一个AI小知识，科普/介绍类，不是路径一的反常识打法。

---

## 固定结构（三段式）

### 一、开头（固定句式）

```
每天分享一个AI知识，[知识点名称]是什么？
```

> 这句话固定不变，只替换知识点名称。

### 二、正文（大白话讲清楚）

- 用最通俗易懂的大白话解释这个知识点
- 必须举例子，让人一听就懂
- 例子要贴近目标用户的日常（宝妈、上班族、大学生能想象到的场景）
- 不用技术词，不用反常识，不用泼冷水，就是**讲明白**

### 三、收尾（说明有什么用）

- 这个小知识对普通人有什么实际用处
- 落点：能帮你赚钱、省时间、或者避坑

---

## 语气要求

- 像朋友聊天，不像上课
- 平和、亲切、好懂
- **不用**路径一的狠话（"屁用没有""大坑"）
- **不用**反常识开头
- **不用**情绪词收尾

---

## 示例骨架

```
每天分享一个AI知识，提示词是什么？

你用ChatGPT或者Claude的时候，打字问它问题，你打的那句话就叫提示词。
比如说你跟AI说"帮我写一条朋友圈文案"，这句话就是提示词。
但如果你说"我是卖水果的，帮我写一条朋友圈文案，要接地气，带上今天榴莲到货了"，
这也是提示词，但效果比前面那句好十倍。
简单说，提示词就是你跟AI说话的方式，你说得越清楚，AI给你的东西越好用。

知道这个有什么用呢？
你以后用AI帮你干活，不管是写文案还是做表格，
把你的需求说具体——你是干什么的、要给谁看、要什么风格，
出来的东西直接就能用，不用反复改，省的就是真金白银的时间。
```

---

## 路径二 vs 路径一

| | 路径一（推文转口播） | 路径二（AI小知识） |
|--|---|---|
| 开头 | 反常识钩子 | "每天分享一个AI知识，XX是什么？" |
| 正文 | 泼冷水→揭真相→案例 | 大白话解释+举例子 |
| 收尾 | 情绪词收尾 | 说明这个知识对你有什么用 |
| 语气 | 重、敢说 | 平和、亲切、好懂 |
| 目的 | 制造认知冲击 | 讲明白一个东西 |
