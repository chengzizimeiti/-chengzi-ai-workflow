# 结构模板库

> 脚本的骨架结构。选好结构后，从其他元素库填充内容。

---

## 结构一：反常识单案例型（主力结构）

```
[0-3秒]  反常识钩子（从钩子库选）
[3-8秒]  解释为什么大家会这么想（建立共鸣）
[8-15秒] 揭露真相/泼冷水（用画面感表达）
[15-40秒] 1个核心案例（从案例库选，带具体数字）
[40-50秒] 说出用户真实处境（从金句库选共鸣句）
[50-55秒] 具体行动第一步（工具名+操作）
[55-60秒] 情绪词收尾 + 预告下期
```

**已用**：2026-03-11、2026-03-12 脚本

---

## 结构二：对比翻转型

```
[0-3秒]  反常识钩子
[3-15秒] 展示"错误做法"（让用户对号入座）
[15-20秒] 给这个错误命名（"我管这个叫XXX"）
[20-40秒] 展示"正确做法" + 案例
[40-50秒] 对比总结（一句话概括差异）
[50-60秒] 行动指引 + 情绪收尾
```

**已用**：2026-03-12 脚本（部分使用）

---

## 结构三：路径二科普三段式

```
[0-3秒]   开头固定句："每天榨干一个AI知识，XX是什么？"
[3-20秒]  共鸣提问 + 大白话解释（不用技术词）
[20-40秒] 生活类比 / 具体案例（贴近宝妈/上班族场景）
[40-55秒] 配套录屏素材（可100%复现的操作步骤）
[55-60秒] 说明实际用处（赚钱/省时间/避坑）
[最后]    结尾固定句："关注橙子，每天榨一点AI"
```

**已用**：2026-03-12 Claude项目功能、AI幻觉 脚本
