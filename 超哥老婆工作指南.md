# 超哥老婆工作指南

> 橙子专用，老公写的，照着做就行，不用懂技术。

---

## 第零步：打开工具

每次工作前，打开电脑上的 **终端**（Terminal），输入下面这行命令，按回车：

```
cd ~/橙子工作流 && claude
```

> 如果超哥帮你设置好了路径，直接输 `claude` 就行。
> 看到 Claude 跟你打招呼了，就说明成功了，可以开始干活。

---

## 你每天要做的事（就3件）

### 一、拆素材（刷到好内容的时候）

你在推特、抖音、小红书刷到好的推文或者同行爆款脚本，**直接复制粘贴到对话里**就行。

Claude 会自动帮你：
1. 判断这条内容跟咱们账号相不相关
2. 把里面好用的东西拆出来给你看（钩子怎么写的、案例怎么讲的、什么金句好用）

然后你来决定：
- 觉得好的，回复 **"入库"** → Claude 自动存起来
- 觉得不行的，回复 **"不要"** → 丢掉
- 今天不想用但以后可能用，回复 **"先存着"** → 放到待用素材库

**举个例子：**

```
你：（粘贴一条推文内容）

Claude：这条关联度高，我拆解出以下元素——
  钩子：……
  案例：……
  金句：……
  你看哪些要入库？

你：钩子和案例入库，金句不要

Claude：好的，已入库。
```

就是这么简单，**粘贴 → 看 → 说入库或不要**，三步。

---

### 二、出脚本（要发视频的时候）

直接跟 Claude 说：

```
帮我出一条脚本，选题是：XXXXX
```

或者：

```
把这条推文转成脚本：（粘贴推文）
```

Claude 会从你之前积累的元素库里挑合适的钩子、案例、金句，按你的风格组合成脚本。

**脚本出来后你要做的：**
1. 读一遍，看看顺不顺口
2. 哪里不对劲，直接改（改完粘贴回来给 Claude 看）
3. Claude 会学习你的修改，下次写得更像你

---

### 三、跟老公同步（重要！）

#### 看老公给你更新了什么

超哥在他电脑上优化了工作流之后，你只需要说：

```
查看老公更新
```

Claude 会自动把最新版拉过来，还会用中文告诉你老公改了什么。

#### 把你的改动推给老公看

如果你改了脚本、加了素材，想让老公看看，说：

```
请老公查阅
```

Claude 会自动把你的改动推过去。超哥在他那边就能看到了。

## 注意事项

1. **推文不要发链接**，Claude 打不开。直接复制文字粘贴进来就行
2. **截图也可以**，直接拖进来或者粘贴图片
3. **一次别喂太多**，一条一条来，拆完一条再发下一条
4. **改完脚本记得给 Claude 看**，它会学习你的风格，越用越像你
5. **不懂就问**，直接问 Claude "这个怎么用"，它会教你

---

## 出了问题怎么办

| 情况 | 怎么办 |
|------|--------|
| Claude 没反应 | 关掉终端重新打开，再输 `claude` |
| 拉不到老公的更新 | 跟老公说一声，可能是他还没推 |
| 不小心删了什么东西 | 别慌，跟老公说，git 能恢复 |
| 看不懂 Claude 说的话 | 直接说"说人话"，它会换个方式解释 |

---

> 有任何问题找老公就行，系统的事他来搞，你只管创作 💪
